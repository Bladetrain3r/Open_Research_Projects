# Results Directory

This directory stores research results, visualizations, and output.

## Structure

- **figures/**: Generated plots, charts, and visualizations
- **tables/**: Result tables and summary statistics

## Organization

### Figures

Store all generated visualizations:
- Charts and graphs
- Plots and diagrams
- Infographics
- Use descriptive filenames (e.g., `correlation_plot_2024.png`)

### Tables

Store tabular results:
- Summary statistics
- Comparison tables
- Result matrices
- Use CSV or Markdown format when possible

## Best Practices

1. **File Naming**: Use clear, descriptive names
   - Include date or version: `results_v1_2024-01-01.csv`
   - Describe content: `mean_variance_by_group.csv`

2. **Reproducibility**: 
   - Include scripts that generated each result
   - Document parameters and settings used
   - Version control result files or note when they change

3. **Formats**:
   - Figures: PNG, SVG, or PDF
   - Tables: CSV, JSON, or Markdown
   - Reports: Markdown or PDF

4. **Documentation**:
   - Add captions and descriptions
   - Note data sources and methods
   - Include interpretation where relevant

## Updating Results

When updating results:
1. Re-run analysis scripts to regenerate
2. Verify outputs match expectations
3. Update related documentation
4. Commit changes with descriptive messages
